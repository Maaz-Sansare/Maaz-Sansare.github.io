Detailed Description

The "Ring buffer" entity is a fixed-size data structure that efficiently manages a sequence of bytes with a wrap-around behavior.

In a ring buffer, the bytes are arranged sequentially in memory with a fixed size and can be accessed using provided apis.

Note
Ring buffer is not thread-safe. If multiple threads access and modify the same Ring buffer instance concurrently then appropriate synchronization mechanisms should be implemented by the user to ensure thread safety.
Input validation is done only when package is built in debug mode.

  • PCF_S_INVALID_ARG : config.size is 0. (Checked only in debug mode)
  • PCF_S_NULL_ARG : config is PCF_NULL. (Checked only in debug mode)

  • On failure - PCF_NULL.
