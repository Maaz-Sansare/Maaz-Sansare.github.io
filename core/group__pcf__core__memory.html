PCF_CORE_API pcf_status_t pcf_Memory_move ( pcf_generic_t  dest,
pcf_generic_t  src,
pcf_size_t  size 
)

Copies size bytes from the location pointed by src to the memory block pointed by dest.

The copy is not done directly the data from src is copied into a temporary buffer and then copied into dest. Hence, even if src and dest overlap there will be no problem.

Parameters
dest- Pointer to the destination memory where the content is to be copied.
src- Pointer to the source memory from where data is to be copied.
size- Number of bytes to copy.
Returns
pcf_status_t valid values :
  • PCF_S_SUCCESS
  • PCF_S_NULL_ARG
  • PCF_S_INVALID_ARG
