Detailed Description

The "linear buffer" entity is a contiguous block of memory that stores a sequence of bytes.

In a linear buffer, the elements are arranged sequentially in memory with a fixed size and can be accessed using provided APIs. The linear buffer acts as a write-ahead buffer, meaning that new elements are written to the buffer in a linear fashion, always advancing the write address. The buffer does not loop back to the beginning when the end is reached without data being read or buffer being flushed.

Note
Linear buffer is not thread-safe. If multiple threads access and modify the same Linear buffer instance concurrently then appropriate synchronization mechanisms should be implemented by the user to ensure thread safety.
Input validation is done only when package is built in debug mode.

  • PCF_S_INVALID_ARG : config.size is 0. (Checked only in debug mode)
  • PCF_S_NULL_ARG : config is PCF_NULL. (Checked only in debug mode)
Returns
  • On success : Pointer to LinearBuffer_t
  • On failure : PCF_NULL
