Detailed Description

The "string" entity provide string functionalities.

Allows user to manipulate string. It includes functions such as finding substring, getting string length, copying, appending, etc.

◆ pcf_String_init()

PCF_CORE_API pcf_status_t pcf_String_init ( const StringConfig_t *const  config)

Initialises the String item to use the specified memory configuration.

Note
The call is optional, and the memory configuration is typically preconfigured to utilize pcf/memory APIs by default.
The String entity requires the following memory APIs: Allocator, Reallocator, Deallocator, Memory Compare, Memory Copy, and Memory Move.
Parameters
config[IN] - String configurations.
Returns
Valid values :
  • PCF_S_SUCCESS : On Success.
  • PCF_S_NULL_ARG : config or config.memAlloc or config.memRealloc or config.memFree or config.memCopy or config.memCompare or config.memMove argument is PCF_NULL.
