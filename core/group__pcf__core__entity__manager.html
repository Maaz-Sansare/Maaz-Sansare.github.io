Initialises the entity manager object.

Parameters
config[IN] - Pointer to EntityManagerConfig_t to set array configurations.
Returns
pcf_status_t Valid values :
  • PCF_ENTITYMANAGER_S_SUCCESS : Successful operation
  • PCF_ENTITYMANAGER_S_NULL_ARG : Null config or config member is passed (Checked only in debug mode)

  • PCF_ENTITYMANAGER_S_ALREADY_CONFIGURED : Entity manager is already configured
