#define PCF_CONDITIONAL_INCLUDE_FILE (   PATH,
  PACKAGE,
  MODULE,
  INCLUDE_NAME 
)     PCF_CONCAT_4(PATH, PACKAGE, MODULE, INCLUDE_NAME)

Concatenates the arguments passed to create a file path that conforms to the path format used in C for include files.

Used for defining the file to be included.

For example including "item_one.h" from "my_package" under scope "my_team":

#define CONDITIONAL_INCLUDE PCF_CONDITIONAL_INCLUDE_FILE(my_team/,my_package/,item_one/,item_one.h)
#include CONDITIONAL_INCLUDE
#undef CONDITIONAL_INCLUDE

CONDITIONAL_INCLUDE = my_team/my_package/item_one/item_one.h

Parameters
PATH- Directory of the project.
PACKAGE- Package name.
MODULE- Package_Item name.
INCLUDE_NAME- File name.
Returns
Concatenated file path.
