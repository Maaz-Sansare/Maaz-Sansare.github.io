Introduction

Configuration package @pcf/config is a required package to be used by other PCF C packages and applications using PCF C packages.

This package defines build configuration macros and common types depending on the platform, i.e. architecture, OS, language and compiler detected.

How to use

To use the package include the pcf/config/config.h in your application or package. To configure the package you need to define the following compile macros while building your application/package

PCF_CONF_LANGUAGE - Valid values c or cpp
PCF_CONF_LANGUAGE_VERSION - Valid values c11, c89, c99 or c17
PCF_CONF_ARCH_DEVICE_VENDOR - Valid values stm, nxp or infineon
PCF_CONF_ARCH_DEVICE_SERIES - Valid values stm32f3, stm32f4, lpc1700 or xmc4000
PCF_CONF_ARCH_DEVICE - Valid values stm32f302r8, stm32f405rg, lpc1768 or xmc4700

Note
Macros which start with "__" or "_" and doesn't end with "__" are reserved to support internal infrastructure and not meant to be used in your code.
