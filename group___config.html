Detailed Description

Configuration package to be used by other PCF C packages and applications using PCF C packages. This package sets up the build configurations and common types depending on the platform, i.e. architecture, OS, language and compiler detected.

Note
This is a required package for all PCF C packages and applications using PCF C packages. For using the @pcf/config package you should include “pcf/config/config.h” in your package and/or application.
