◆ pcf_Pwm_new()

PCF_MCAL_API Pwm_t * pcf_Pwm_new ( const PwmConfig_t *const  config,
pcf_status_t *  status 
)

Create a PWM Entity Object.

It creates and initializes PWM Entity according to the configuration structure PwmConfig_t and returns a pointer to Pwm_t.

Parameters
config[IN] - Pointer to the pwm configuration structure.
status[OUT] -
  • PCF_PWM_S_SUCCESS : Successful operation.
  • PCF_PWM_S_NULL_ARG : Null argument was passed.(Checked only in debug mode)
  • PCF_PWM_S_INVALID_ARG : Invalid config member value was passed.(Checked only in debug mode)
  • PCF_PWM_S_ALREADY_IN_USE : Specified pwm is already in use.
  • PCF_PWM_S_ALLOCATION_FAILED : Memory allocation failed.
Returns
Pointer of the Pwm_t object or PCF_NULL in case of error.
