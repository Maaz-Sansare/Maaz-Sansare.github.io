Detailed Description

Periodic Timer config structure.

Example to configure PeriodicHWTimer_t: PeriodicHWTimerConfig_t config = {0}; config.id = PCF_PERIODIC_TIMER_0; // Id 0 config.eventTime.value = 200; config.eventTime.unit = enTimeUnit_us; // 200us config.hardwareCfg = PCF_NULL; // No hardware specific config
