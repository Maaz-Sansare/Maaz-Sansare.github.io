Detailed Description

Pwm config structure.

Example to configure PWM:

PwmConfig_t config = {0};
config.portPin.port = PCF_PORT_0; // Port 0
config.portPin.pin = PCF_PIN_2; // Pin 2
config.period.value = 200;
config.period.unit = enTimeUnit_us; // 200us
config.outputSignal = enPwmOutputSignal_Normal;
config.hardwareCfg = PCF_NULL; // No hardware specific config
