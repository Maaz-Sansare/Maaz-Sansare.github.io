◆ pcf_Peripheral_configure()

PCF_MCAL_API pcf_status_t pcf_Peripheral_configure ( const PeripheralConfig_t *const  config)

Configures the peripheral package to use the specified configuration.

Note
The call is optional, and the memory configuration is typically preconfigured to utilize pcf/memory(@core pkg) APIs by default.
The Peripheral APIs mandate the usage of the following memory APIs: Allocator, Reallocator, and Deallocator.
Parameters
config[IN] - The peripheral configuration.
Returns
pcf_status_t Valid values :
  • PCF_PERIPHERAL_S_SUCCESS : On success.
  • PCF_PERIPHERAL_S_NULL_ARG : config or config.memAlloc or config.memRealloc or config.free is PCF_NULL.(Checked only in debug mode)
  • PCF_PERIPHERAL_S_ALREADY_CONFIGURED : pcf_Peripheral_init is already configured.
