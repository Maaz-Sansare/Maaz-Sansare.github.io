Detailed Description

Periodic Timer config structure.

Example to configure PERIODICTIMER: PeriodicTimerConfig_t config = {0}; config.id = PCF_PERIODIC_TIMER_0; // Id 0 config.eventTime.value = 200; config.eventTime.unit = enTimeUnit_us; // 200us // Must include <pcf/memory/memory.h> in order to utilize the @pcf/core memory APIs. config.memoryConfig.memAlloc = pcf_Memory_allocate; // Using @pcf/core memory allocater. config.memoryConfig.memFree = pcf_Memory_free; // Using @pcf/core memory free. config.hardwareCfg = PCF_NULL; // No hardware specific config
