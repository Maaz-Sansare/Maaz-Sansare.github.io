Detailed Description

QEI configuration structure.

Example to configure QEI:

QeiConfig_t config = {0};
config.chnlA.port = PCF_PORT_0; // Port 0
config.chnlA.pin = PCF_PIN_0; // Pin 0
config.chnlB.port = PCF_PORT_0; // Port 0
config.chnlB.pin = PCF_PIN_2; // Pin 2
config.cpr = 4; // Counts per revolution
config.encoding = enQeiResolution_16; // 16 bit Resolution.
// Must include <pcf/memory/memory.h> in order to utilize the @pcf/core memory APIs.
config.memoryConfig.memAlloc = pcf_Memory_allocate; // Using @pcf/core memory allocater.
config.memoryConfig.memFree = pcf_Memory_free; // Using @pcf/core memory free.
config.hardwareCfg = PCF_NULL; // No hardware specific config
