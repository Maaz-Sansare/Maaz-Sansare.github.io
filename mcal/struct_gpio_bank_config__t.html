Detailed Description

GpioBank configuration structure.

Example to configure GPIOBANK:

GpioBankConfig_t config = {0};
config.portFirstPin.port = PCF_PORT_0; // Port 0
config.portFirstPin.pin = PCF_PIN_0; // Pin 0
config.size = 3; // Size is 3
config.direction = enGpioDirection_Output; // Direction Output.
config.pupd = enGpioPUPD_None; // No Pull up or Pull down i.e. pin is floating
config.oType = enGpioOutputType_PushPull; // Output type is Push-Pull
config.speed = enGpioSpeed_High; // The slew rate or transition speed is set to High.
// Must include <pcf/memory/memory.h> in order to utilize the @pcf/core memory APIs.
config.memoryConfig.memAlloc = pcf_Memory_allocate; // Using @pcf/core memory allocater.
config.memoryConfig.memFree = pcf_Memory_free; // Using @pcf/core memory free.
config.hardwareCfg = PCF_NULL; // No hardware specific config

EnGpioOutputType_t GpioBankConfig_t::oType

Gpio Bank Pins output type. Push-Pull or Open-Drain. The value can be set even if direction is Input, However the configuration comes into effect only when direction is Output If direction is Input and this value is not set then by default Output Type is set as Push-Pull

◆ speed

EnGpioSpeed_t GpioBankConfig_t::speed

Gpio Bank Pins transition speed/slew rate. Time taken to go from one state to another. The value can be set even if direction is Input, However the configuration comes into effect only when direction is Output If direction is Input and this value is not set then by default Speed is set as Low
