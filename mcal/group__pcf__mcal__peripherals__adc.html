PCF_MCAL_API Adc_t * pcf_Adc_new ( const AdcConfig_t *  config,
pcf_status_t *  status 
)

This function is used to create a new Adc_t Object.

It creates and initializes Adc_t according to the configuration structure and returns a pointer to Adc_t.

Parameters
config[IN] - Pointer to the adc configuration structure.
status[OUT] -
  • PCF_ADC_S_SUCCESS : Successful operation.
  • PCF_ADC_S_NULL_ARG : Null config or config member has null. (Checked only in debug mode)
  • PCF_ADC_S_INVALID_ARG : Invalid ADCs id or resolution or invalid frequency was passed. (Checked only in debug mode)
  • PCF_ADC_S_ALREADY_IN_USE : Specified id is already in use.
  • PCF_ADC_S_ALLOCATION_FAILED : Memory allocation failed.
Returns
Pointer of the Adc_t object or PCF_NULL in case of error.

)

Add a channel into a ADC group.

If a particular channel is added to a group (regular or injected) it cannot be added to another group.

Parameters
adc[IN] - Pointer to Adc_t object.
channel_id[IN] - Channel id.
group[IN] - The group to which the channel should be added. Regular or Injected.
sample_time[IN] - The sampling time in number of clock cycles.
Returns
pcf_status_t Valid values :
  • PCF_ADC_S_SUCCESS : Successful operation.
  • PCF_ADC_S_NULL_ARG : Adc_t object was PCF_NULL. (Checked only in debug mode)
  • PCF_ADC_S_INVALID_ARG : Invalid Adc_t object or channel_id (or group (based on uC)) was passed. (Checked only in debug mode)
  • PCF_ADC_S_ALREADY_IN_USE : Specified channel_id is already in use.
