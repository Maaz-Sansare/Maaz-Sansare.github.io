Detailed Description

DAC refers to digital to analog converter which is a device that converts digital values into analog voltage signals,
enabling the MCU to interface with analog sensors, displays, and other analog devices.

Note
Input validation is done only when package is built in debug mode.

◆ pcf_Dac_new()

PCF_MCAL_API Dac_t * pcf_Dac_new ( const DacConfig_t *const  config,
pcf_status_t *const  status 
)

Creates a new instance of Dac_t based on the configuration provided by DacConfig_t config .

Parameters
config[IN] - Pointer to the DAC configuration structure DacConfig_t.
status[OUT] - Pointer to pcf_status_t, stores the status of the function call.
(Optional parameter, pass PCF_NULL if not required.)
Valid values :
  • PCF_S_SUCCESS : On success.
  • PCF_S_NULL_ARG : config is PCF_NULL. (Checked only in debug mode)
  • PCF_S_INVALID_ARG : config.id or config.resolution is invalid. (Checked only in debug mode)
  • PCF_S_ALREADY_IN_USE : DAC id is already in used or the mcu pin is already in use.
  • PCF_S_NOT_ENOUGH_MEMORY :Not enough memory to create the Dac entity.
Returns
valid values:
  • On success - Pointer to Dac_t.
  • On failure - PCF_NULL.
