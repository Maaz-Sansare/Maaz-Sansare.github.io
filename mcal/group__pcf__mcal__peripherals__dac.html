◆ pcf_Dac_new()

PCF_MCAL_API Dac_t * pcf_Dac_new ( const DacConfig_t *const  config,
pcf_status_t *const  status 
)

Object specific functions.

Creats a new DAC entity object. It creates and initializes DAC according to the configuration structure and returns a pointer to Dac_t.

Parameters
config[IN] - Pointer to the dac configuration structure.
status[OUT] - Pointer to pcf_status_t Valid values :
  • PCF_DAC_S_SUCCESS : Successful operation.
  • PCF_DAC_S_NULL_ARG : Null config is passed. (Checked only in debug mode)
  • PCF_DAC_S_INVALID_ARG : Invalid config.id or config.resolution is passed. (Checked only in debug mode)
  • PCF_DAC_S_ALREADY_IN_USE : DAC id is already in used or the mcu pin is already in use.
  • PCF_DAC_S_ALLOCATION_FAILED : Memory allocation failure.
Returns
On success, it return the pointer to Dac_t DAC device. On error, it returns NULL.
