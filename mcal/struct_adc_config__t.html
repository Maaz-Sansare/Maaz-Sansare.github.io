Detailed Description

ADC config struct.

Example to configure ADC:

AdcConfig_t config = {0};
config.id = PCF_ADC_1; // Adc id 1
config.referenceVoltage = 3300; // 3.3v
config.resolution = 8; // 8 bit resolution.
config.frequency.value = 5;
config.frequency.unit = enFrequencyUnit_MHz; // 5MHz
config.hardwareCfg = PCF_NULL; // No hardware specific config
