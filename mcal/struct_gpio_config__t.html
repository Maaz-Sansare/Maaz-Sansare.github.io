Detailed Description

Gpio configuration structure.

Example to configure GPIO:

GpioConfig_t config = {0};
config.portPin.port = PCF_PORT_0; // Port 0
config.portPin.pin = PCF_PIN_2; // Pin 2
config.direction = enGpioDirection_Output; // Direction Output.
config.pupd = enGpioPUPD_None; // No Pull up or Pull down i.e. pin is floating
config.oType = enGpioOutputType_PushPull; // Output type is Push-Pull
config.speed = enGpioSpeed_High; // The slew rate or transition speed is set to High.
config.hardwareCfg = PCF_NULL; // No hardware specific config

EnGpioOutputType_t GpioConfig_t::oType

Gpio output type. Push-Pull or Open-Drain. The value can be set even if direction is Input, However the configuration comes into effect only when direction is Output. If direction is Input and this value is not set then by default Output Type is set as Push-Pull

◆ speed

EnGpioSpeed_t GpioConfig_t::speed

Gpio transition speed/slew rate. Time taken to go from one state to another. The value can be set even if direction is Input, However the configuration comes into effect only when direction is Output If direction is Input and this value is not set then by default Speed is set as Low
