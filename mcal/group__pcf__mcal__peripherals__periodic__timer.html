◆ pcf_PeriodicTimer_attachCallback()

PCF_MCAL_API pcf_status_t pcf_PeriodicTimer_attachCallback ( PeriodicTimer_t *const  p_timer,
const PeriodicTimerCallback_fp  callback,
const pcf_generic_t  user_arg,
PeriodicTimerCallback_fp *const  old_callback,
pcf_generic_t *const  old_userArg 
)

Attaches a callback, to be called at a recurring interval with which the periodic timer was configured.

Note
Call this function before starting the periodic timer. To attach a callback if periodic timer has already started:
  1. Stop the periodic timer using pcf_PeriodicTimer_stop.
  2. Reset the periodic timer pcf_PeriodicTimer_reset.
  3. Detach the callback pcf_PeriodicTimer_detachCallback.
  4. Start the periodic timer again pcf_PeriodicTimer_start.
Parameters
p_timer[IN] - Pointer to periodic timer entity object.
callback[IN] - User callback to be called at the specified compare_time
user_arg[IN] - User data that will be passed to the callback. Can be set to NULL.
old_callback[OUT] - The callback previously registered. Will be set to NULL if no callback was registered

Optional parameter, pass PCF_NULL if you do not want the callback previously registered.

Parameters
old_userArg[OUT] - The previous user data set.

Optional parameter, pass PCF_NULL if you do not want the older user defined data.

Returns
pcf_status_t Valid values:
  • PCF_PERIODIC_TIMER_S_SUCCESS : Successful operation.
  • PCF_PERIODIC_TIMER_S_NULL_ARG : Null p_timer pointer or null callback function pointer is passed (Checked only in debug mode)

  4. Start the periodic timer again using pcf_PeriodicTimer_start.
