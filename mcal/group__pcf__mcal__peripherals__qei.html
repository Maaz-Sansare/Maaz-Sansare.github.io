◆ pcf_Qei_new()

PCF_MCAL_API Qei_t * pcf_Qei_new ( const QeiConfig_t *  config,
pcf_status_t *  status 
)

Create a qei entity.

Call this function in beginning to initialize the qei entity.

This function allocates memory for the entity using pcf_malloc() function call and returns a pointer to the entity. Pass this entity pointer while calling qei functions.

Parameters
config[IN] - Pointer to the qei configuration structure.
status[OUT] -
  • PCF_QEI_S_SUCCESS : Successful operation.
  • PCF_QEI_S_NULL_ARG : Null argument was passed.(Checked only in debug mode)
  • PCF_QEI_S_INVALID_ARG : Invalid config member value was passed.(Checked only in debug mode)
  • PCF_QEI_S_ALREADY_IN_USE : Specified pwm is already in use.
  • PCF_QEI_S_ALLOCATION_FAILED : Memory allocation failed.
Returns
Pointer of the Qei_t object or PCF_NULL in case of error.
