Detailed Description

QEI refers to Quadrature Encoder Interface is a specialized hardware feature that enables microcontrollers to interface with rotary encoders, accurately measuring angular position and direction.

QEI is commonly used in robotics and industrial automation for tasks like tracking wheel movement and determining precise motor positions.

Note
Input validation is done only when package is built in debug mode.
