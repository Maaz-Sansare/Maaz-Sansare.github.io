Detailed Description

RTC refers to Real-Time Clock is an essential component that provides accurate timekeeping and calendar functionality to microcontrollers, allowing them to track time and date independently of external power sources.

RTCs are commonly used in applications such as scheduling tasks, logging events, and maintaining accurate timestamps in embedded systems.

Note
Input validation is done only when package is built in debug mode.
