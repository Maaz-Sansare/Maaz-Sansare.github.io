◆ EnMcuResetReason_t

enum EnMcuResetReason_t

MCU last reset reason.

Enumerator
enMcuResetReason_PowerOn 

Set when power is initially applied to the board/CPU

enMcuResetReason_Reset 

Set when a reset is triggered by a hardware pin on the board/CPU

enMcuResetReason_BrownOut 

Set when a reset is triggered by brown-out i.e. when the supply voltage falls below a certain threshold

enMcuResetReason_Software 

Set when a reset is triggered from software

enMcuResetReason_Watchdog 

Set when a reset is triggered due to watchdog timer refresh failure

enMcuResetReason_Lockup 

Set when a reset is triggered due to lockup

enMcuResetReason_Wakeup 

Set when cpu wakes up from deep sleep mode on low power

enMcuResetReason_AccessError 

Set when any access related issue causes reset

enMcuResetReason_Unknown 

Unknown reason for reset
