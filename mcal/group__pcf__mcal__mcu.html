◆ EnMcuPowerMode_t

enum EnMcuPowerMode_t

MCU power mode enum.

Enumerator
enMcuPowerMode_Sleep 

Mcu core clock is powered down and switched to the internal clock (IRC) Resumption can be done via external interrupt, RTC alarm, Watch dog, etc

enMcuPowerMode_DeepSleep 

Same as enMcuPowerMode_Sleep + also disable flash memory

enMcuPowerMode_StandBy 

Mcu core clock is slow down and unused peripheral clock is stopped.

◆ EnMcuResetReason_t

enum EnMcuResetReason_t

MCU last reset reason.

Enumerator
enMcuResetReason_PowerOn 

Set when power is initially applied to the board/CPU

enMcuResetReason_Reset 

Set when a reset is triggered by a hardware pin on the board/CPU

enMcuResetReason_BrownOut 

Set when a reset is triggered by brown-out i.e. when the supply voltage falls below a certain threshold

enMcuResetReason_Software 

Set when a reset is triggered from software

enMcuResetReason_Watchdog 

Set when a reset is triggered due to watchdog timer refresh failure

enMcuResetReason_Lockup 

Set when a reset is triggered due to lockup

enMcuResetReason_Wakeup 

Set when cpu wakes up from deep sleep mode on low power

enMcuResetReason_AccessError 

Set when any access related issue causes reset

enMcuResetReason_Unknown 

Unknown reason for reset

PCF_MCAL_API pcf_status_t pcf_Mcu_setPowerMode ( const EnMcuPowerMode_t  mode,
const EnMcuWaitFor_t  waitFor,
const EnPCFBoolean_t  sleepOnExit,
const McuCallback_fp  eventHandler,
const pcf_generic_t  arg 
)

Sets the MCU to power mode based on specified value.

Parameters
mode[IN] - Power mode.
waitFor[IN] - Wakeup condition i.e. interrupt or event.

.

Parameters
sleepOnExit[IN] - If enPCFBoolean_True, mcu will again go in power down mode after returning from ISR handler.
eventHandler[IN] - Function pointer to the mcu callback function. (Optional parameter, pass PCF_NULL if not required.)
arg[IN] - User-defined arguments to be passed to the callback function. (Optional parameter, pass PCF_NULL if not required.)
Returns
pcf_status_t Valid values :
  • PCF_MCU_S_SUCCESS : On success.
  • PCF_MCU_S_INVALID_ARG : mode or waitFor is invalid.(Checked only in debug mode)
