◆ pcf_GpioBank_new()

PCF_MCAL_API GpioBank_t * pcf_GpioBank_new ( const GpioBankConfig_t *const  config,
pcf_status_t *  status 
)

This function is used to create a GpioBank Entity Object.

It creates and initializes GpioBank Entity according to the configuration structure GpioBankConfig_t and returns a pointer to GpioBank_t.

Parameters
config[IN] - Pointer to the gpio bank configuration structure.
status[OUT] -
  • PCF_GPIO_BANK_S_SUCCESS : Successful operation.
  • PCF_GPIO_BANK_S_NULL_ARG : Null argument was passed.(Checked only in debug mode)
  • PCF_GPIO_BANK_S_INVALID_ARG : Invalid port,pin,direction,output type or speed was passed.(Checked only in debug mode)
  • PCF_GPIO_BANK_S_ALREADY_IN_USE : Specified port pin is already in use.
  • PCF_GPIO_BANK_S_ALLOCATION_FAILED : Memory allocation failed.
Returns
Pointer of the GpioBank_t object or PCF_NULL in case of error.

◆ pcf_GpioBank_write()

PCF_MCAL_API pcf_status_t pcf_GpioBank_write ( const GpioBank_t *const  gpio_bank,
const uint32_t  value 
)

Writes a value to the gpio bank (This can only be used when gpio bank direction is set to Output).

Parameters
gpio_bank[IN] - Pointer to GpioBank_t object.
value[IN] - An integer where each bit of the integer corresponds to an associated port pin. If bit is high the pin is set if bit is low the pin is cleared. The LSb is mapped to the starting pin of the Bank. bit 0 of value -> First Pin in the Bank bit 1 of value -> Second Pin in the Bank bit N of value -> Nth Pin in the Bank.
Returns
pcf_status_t Valid values :
  • PCF_GPIO_BANK_S_SUCCESS : Successful operation.
  • PCF_GPIO_BANK_S_NULL_ARG : Null argument was passed.(Checked only in debug mode)
  • PCF_GPIO_BANK_S_INVALID_OPERATION : Gpio direction is not in output mode.
